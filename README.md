# Loan-Approval-Credit-Risk-Analysis
End-to-end business analytics project analyzing loan approval patterns and predicting loan approval using machine learning.

Banks need to evaluate loan applications while minimizing financial risk.

This project analyzes applicant characteristics to identify the factors influencing loan approval and develops a machine learning model capable of predicting approval decisions.

✔ Perform Exploratory Data Analysis

✔ Identify factors affecting approval

Key Findings

Average CIBIL score of approved applicants was significantly higher than rejected applicants.
Education had minimal influence on loan approval rates.
Loan approval was strongly associated with creditworthiness.
Predictive model successfully classified loan applications.

Business Recommendations

Banks should continue prioritizing applicants with strong credit scores while considering additional financial indicators rather than relying heavily on educational qualifications.
